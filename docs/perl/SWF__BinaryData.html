NAME

SWF::BinaryData - SWF binary data class

SYNOPSIS

        use SWF::BinaryData;
        $binarydata = new SWF::BinaryData($data, $length);

DESCRIPTION

SWF::BinaryData is a helper class useful in ActionScript.
